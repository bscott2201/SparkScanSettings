Requirements for Scandit Data Capture SDK 6.24

Native Platforms

Platform

Requirements

Android

  • Android 6.0 (API level 23)

  • Java 8

  • ARMv7, ARM64, x86, x86_64

  • Camera resolution of 720p or higher is recommended

  • GPU required 1

iOS

  • iOS 13.0

  • ARM64, x86_64

  • Camera resolution of 720p or higher is recommended

  • GPU required
1

On Android hardware acceleration has to be enabled for the camera preview to work. Hardware acceleration is enabled by default since API level 14 but can potentially be turned off in the Manifest, do not do this if you use the camera.

Web Platforms

Requirements

  • Blob (supported browser versions)

  • URL/createObjectURL (supported browser versions)

  • Web Workers (supported browser versions)

  • WebAssembly (supported browser versions)

  • SharedArrayBuffer (supported browser versions) - required for simultaneous multiple barcode tracking.

  • MediaDevices/getUserMedia (supported browser versions) - optional, required for camera video streaming

  • OffscreenCanvas (supported browser versions) - optional, allows for GPU acceleration in Web Worker

  • WebGL (supported browser versions) - optional, allows for GPU acceleration in Web Worker

  • WebAssembly SIMD - optional, allows for OCR computation acceleration
